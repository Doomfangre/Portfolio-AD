EECU App

EECU Budgeting App

Jan 2026 - Mar 2026

I was tasked with creating an app for the official EECU and their consumers. It was a budget calculator that had special features such as giving tips on budgeting and how much they are spending over budget. 

The Technology Stack

    • HTML5, CSS3, and Vanilla JavaScript.
      Hosted via Netlify to ensure global accessibility and performance.
      Visual Studio Code was used for the coding platform and Github was used for handling the code.

    • Concepts Learned

    • I used HTML, CSS, and mostly JavaScript for the coding and it was all done in visual studio code. It was mostly made up of img>, div>, and span>.
      The website was responsive enough to become a phone screen.
      This project enlightened me on more HTML and JavaScript knowledge I did not know previously.

    • Challenges & Solutions

    • The Problem: I had a hard time understanding and creating the pie chart to calculate the inputted budget.
      The Solution: There was a website called Chart.js that had all of the information and guides for all types of charts. All I had to do was do what the site was doing.

    • Habits of Mind in Action

    • Reflection: I learned quite a bit from the project post-mortem, like how CSS responsive design was changing depending on whether it was in a grid or a flexbox.
      Communication: I had to communicate with my peers for additional help and work for the project.
